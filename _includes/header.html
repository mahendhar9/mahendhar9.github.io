To scroll to a specific section

  • - Say 'Go to #{section-name}'.

  • - For ex, 'Go to Skills'.

    When viewing Portfolio

  • - Say 'All Projects' to view All Projects.

  • - Say 'HTML Projects' to view all HTML & CSS Projects.

  • - Say 'Javascript Projects' to view all Javascript & Jquery Projects.

  • - Say 'Rails Projects' to view all Ruby on Rails Projects.

  • - Say 'Angular Projects' to view all AngularJS Projects.
